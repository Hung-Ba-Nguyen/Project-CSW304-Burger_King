# 🍔 Burger King - Fast Food Restaurant Website

Một giao diện trang web nhà hàng thức ăn nhanh hiện đại, thân thiện với người dùng. Đây là dự án thực hành Front-end đầu tay của tôi nhằm củng cố kỹ năng dựng giao diện web thực tế dựa trên Bootstrap 5.

## 🔗 Live Demo
> 🌍 **[Xem Live Demo tại đây](https://Hung-Ba-Nguyen.github.io/Project-CSW304-Burger_King/)**

## 🛠 Công nghệ sử dụng
*   **HTML5 & CSS3:** Xây dựng cấu trúc và định dạng giao diện trang web.
*   **JavaScript:** Xử lý các tương tác UI cơ bản.
*   **Framework/Thư viện:** Bootstrap 5 (Responsive grid, UI components), FontAwesome (Icons).

## ✨ Tính năng nổi bật
*   **Responsive Web Design:** Giao diện hiển thị tốt trên nhiều kích thước màn hình (Desktop, Tablet, Mobile) nhờ hệ thống Grid của Bootstrap.
*   **Interactive UI:** Tích hợp Carousel Slider cho banner, các hiệu ứng hover ảnh và giao diện Form đặt bàn (Booking).
*   **Bố cục trực quan:** Phân chia rõ ràng các section như About, Feature, Menu, Contact, Blog.

## 📈 Tự đánh giá & Hướng cải thiện (Lessons Learned)
Vì đây là một trong những dự án Front-end đầu tiên, tôi tập trung chủ yếu vào việc hoàn thiện UI/UX sao cho giống thiết kế nhất. Trong tương lai, tôi dự định sẽ refactor (tái cấu trúc) lại mã nguồn:
*   Tách hoàn toàn các đoạn inline-styles trong HTML ra file CSS độc lập.
*   Áp dụng quy tắc đặt tên chuẩn (như BEM) cho các class/id để code ngắn gọn và dễ bảo trì hơn.
*   Tối ưu hóa lại cấu trúc thẻ tag (Semantic HTML).

## 🚀 Hướng dẫn cài đặt (Local Development)
1. Clone repository này về máy:
   `git clone https://github.com/Hung-Ba-Nguyen/Project-CSW304-Burger_King.git`
2. Mở thư mục bằng trình soạn thảo mã (khuyên dùng VS Code).
3. Sử dụng extension **Live Server** để chạy file `index.html`.
