# 🍔 Burger King - Fast Food Restaurant Website

Một giao diện trang web nhà hàng thức ăn nhanh hiện đại, tập trung vào trải nghiệm người dùng (UX) và tối ưu hóa hiển thị trên mọi thiết bị. Dự án này được xây dựng nhằm mục đích củng cố kiến thức nền tảng về Front-end và thực hành viết Clean Code.

## 🔗 Live Demo
> 🌍 **[Xem Live Demo tại đây](https://Hung-Ba-Nguyen.github.io/Project-CSW304-Burger_King/)**

## 🛠 Công nghệ sử dụng
*   **HTML5:** Cấu trúc ngữ nghĩa (Semantic HTML).
*   **CSS3:** Flexbox, Grid, CSS Variables.
*   **JavaScript:** Xử lý sự kiện DOM cơ bản.
*   **Framework/Thư viện:** Bootstrap 5 (Responsive grid, components), FontAwesome (Icons).

## ✨ Tính năng nổi bật & Kỹ năng áp dụng
*   **Responsive Web Design:** Giao diện tương thích hoàn toàn trên Desktop, Tablet và Mobile.
*   **Clean Code & Architecture:** 
    *   Tách biệt hoàn toàn HTML và CSS.
    *   Tối ưu hóa các class CSS, loại bỏ inline-styles để dễ dàng mở rộng và bảo trì.
*   **Interactive UI:** Tích hợp hiệu ứng hover mượt mà, Carousel Slider cho banner.

## 🚀 Hướng dẫn cài đặt (Local Development)
1. Clone repository này về máy:
   `git clone https://github.com/Hung-Ba-Nguyen/Project-CSW304-Burger_King.git`
2. Mở thư mục bằng trình soạn thảo mã (khuyên dùng VS Code).
3. Sử dụng extension **Live Server** để chạy file `index.html`.
